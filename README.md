# 餐食卡路里追蹤應用

一個簡潔易用的餐食卡路里記錄應用，幫助你追蹤每日飲食和卡路里攝入。

## 功能特點

### 📝 記錄餐食
- 快速記錄每一餐的食物和卡路里
- 支援選擇日期和餐別（早餐、午餐、晚餐、小食）
- 從食物資料庫選擇或快速添加臨時食物
- 自動計算總卡路里
- 可添加備註

### 📚 食物資料庫
- 內建常用食物（白飯、雞蛋、麵包等）
- 自訂添加食物及其卡路里
- 標記常用食物（★），方便快速選擇
- 搜尋功能，快速找到需要的食物
- 編輯和刪除食物

### 📖 歷史記錄
- 查看所有餐食記錄，按日期分組
- 顯示每日總卡路里和餐數統計
- 點擊展開/收起每日詳情
- 下拉刷新
- 刪除記錄功能

### 📊 每週總結
- 視覺化圖表顯示每週卡路里攝入
- 週總卡路里、日均卡路里統計
- 上週/下週導航
- 智能洞察和建議
- 每日餐數統計

## 如何使用

### 安裝依賴

```bash
npm install
```

### 啟動應用

```bash
npm start
```

然後選擇：
- 按 `a` 在 Android 模擬器上運行
- 按 `i` 在 iOS 模擬器上運行
- 掃描 QR code 在實體裝置上運行（需要 Expo Go 應用）

## 使用指南

### 第一次使用
1. 啟動應用後，系統會自動初始化 10 種常用食物
2. 你可以在「食物庫」頁面查看和管理這些食物

### 記錄餐食
1. 在「記錄餐食」頁面：
   - 選擇日期和餐別
   - 點擊「從資料庫選擇」添加已有的食物
   - 或點擊「快速添加」添加臨時食物
   - 查看總卡路里
   - 可選添加備註
   - 點擊「保存記錄」

### 管理食物資料庫
1. 在「食物庫」頁面：
   - 點擊「+ 新增食物」添加自訂食物
   - 勾選「設為常用食物 ★」標記為常用
   - 使用搜尋欄快速查找
   - 點擊「編輯」修改食物資訊
   - 點擊「刪除」移除食物

### 查看歷史記錄
1. 在「歷史記錄」頁面：
   - 查看頂部統計卡片
   - 點擊日期卡片展開/收起當日記錄
   - 下拉可刷新資料
   - 點擊「刪除」移除單筆記錄

### 查看每週總結
1. 在「每週總結」頁面：
   - 查看當週統計數據
   - 查看每日卡路里柱狀圖
   - 閱讀智能洞察建議
   - 使用「上週」/「下週」導航按鈕查看其他週

## 資料儲存

所有資料都儲存在應用的本地儲存空間（AsyncStorage），即使關閉應用也不會丟失。

## 技術棧

- **框架**: React Native + Expo
- **路由**: Expo Router
- **本地儲存**: AsyncStorage
- **日期選擇**: DateTimePicker
- **語言**: TypeScript

## 注意事項

- 資料僅儲存在本機，不會上傳到雲端
- 刪除應用會清除所有資料
- 建議定期備份重要記錄（可截圖保存）

## 開發者

此應用使用 Expo 和 React Native 開發，適用於 iOS 和 Android 平台。
